# Data Validation for HART / CHSRMT

This directory contains validation materials used to evaluate the HART
(Heat Assessment and Response Tool) framework.

The validation dataset consists of harmonized field measurements
collected from operational environments including potrooms,
casthouse areas, and outdoor industrial locations.

Variables include:

- Dry bulb temperature (DB)
- Wet bulb temperature (WB)
- Globe temperature (GT)
- Relative humidity (RH)
- Wind speed (WS)
- WBGT
- TWL

These observations were used to compare:

• Thermal Work Limit (TWL)
• Heat-Strain Profile (HSP)

The objective of this validation exercise is to evaluate whether
HSP can provide earlier supervisory warning of physiological
heat strain compared with TWL withdrawal thresholds.
